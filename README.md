# MM-Agent

A simple Anthropic Claude agent built with TypeScript.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Add your Anthropic API key to `.env`:
```
ANTHROPIC_API_KEY=your-actual-api-key-here
```

3. Run the agent:
```bash
npx tsx index.ts
```

## Features

- Basic Claude conversation
- Echo tool functionality
- Environment variable configuration
- TypeScript support with Zod validation
